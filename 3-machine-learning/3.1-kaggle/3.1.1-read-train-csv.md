# 3.1.1 - Kaggle 读取 train.csv

## 一句话

`train = pd.read_csv(os.path.join(path, "train.csv"))`：把 `path` 目录下的 `train.csv` 读成 pandas 的 `DataFrame`；`len(train)` 返回样本行数。

## 最小代码

```python
import os
import pandas as pd

path = "/kaggle/input/titanic"

train = pd.read_csv(os.path.join(path, "train.csv"))
print(len(train))       # 行数
print(train.shape)      # (行数, 列数)
display(train.head())   # 前 5 行
```

## 关键对象

| 对象 | 类型 | 作用 |
| --- | --- | --- |
| `os` | Python 标准库 | 操作路径、目录、文件 |
| `path` | `str` | 数据目录，不是文件 |
| `"train.csv"` | `str` | 文件名 |
| `os.path.join(path, "train.csv")` | `str` | 完整文件路径 |
| `pd` | pandas 别名 | 数据分析库 |
| `pd.read_csv(...)` | 函数 | CSV -> DataFrame |
| `train` | `DataFrame` | 训练集表格 |
| `len(train)` | `int` | DataFrame 行数 |
| `train.shape` | `tuple` | `(行数, 列数)` |

## Kaggle 路径

```text
/kaggle/input/
```

```text
/kaggle/input/titanic/
├── train.csv

```

```python
path = "/kaggle/input/titanic"
```

规则：`path` 指向目录；`train.csv` 是目录里的文件。

```python
os.path.join(path, "train.csv")
# "/kaggle/input/titanic/train.csv"
```

本地项目同理：

```python
path = "./data"
train = pd.read_csv(os.path.join(path, "train.csv"))
```

## 为什么用 `os.path.join`

- 手拼路径容易漏 `/` 或多 `/`
- 不同系统路径分隔符可能不同
- 文件名集中管理，代码更好改

```python
train_path = os.path.join(path, "train.csv")
train = pd.read_csv(train_path)
```

## `pd.read_csv`

CSV 是表格文本；`read_csv` 把它读成 `DataFrame`。

```python
import pandas as pd
train = pd.read_csv("/kaggle/input/titanic/train.csv")
```

`DataFrame` = 二维表：

- 每一行是一条样本
- 每一列是一个字段或特征
- 可以存数字、字符串、日期、类别

```text
PassengerId | Survived | Pclass | Name | Sex | Age
```

- `Survived`：标签，模型要预测的目标
- `Pclass`、`Sex`、`Age` 可能是特征
- 每一行代表一名乘客

## 读取后必查

```python
print("files:", os.listdir(path))
print("rows:", len(train))
print("shape:", train.shape)
print("columns:", train.columns.tolist())
display(train.head())
```

含义：

- `os.listdir(path)`：确认文件真实存在
- `len(train)`：样本数
- `train.shape`：行列规模
- `train.columns`：字段名
- `train.head()`：前 5 行，检查是否读歪

## 完整起手模板

```python
import os
import pandas as pd

path = "/kaggle/input/titanic"

print(os.listdir(path))

train_path = os.path.join(path, "train.csv")
test_path = os.path.join(path, "test.csv")

train = pd.read_csv(train_path)
test = pd.read_csv(test_path)

print("train:", train.shape)
print("test:", test.shape)
display(train.head())
display(test.head())
```

普通 `.py` 文件没有 `display` 时：

print(train.head())
```

## 常见坑

| 坑 | 错误写法 | 结果 | 修正 |
| --- | --- | --- | --- |
| `path` 写成文件 | `path="/kaggle/input/titanic/train.csv"` | 拼成 `.../train.csv/train.csv` | `path="/kaggle/input/titanic"` |
| 文件名大小写错 | `"train.csv"` vs `Train.csv` | `FileNotFoundError` | `os.listdir(path)` 看真实文件名 |
| 忘导入 pandas | `pd.read_csv(...)` | `NameError: pd` | `import pandas as pd` |
| 当前目录误判 | `pd.read_csv("train.csv")` | 本地找不到文件 | 写完整路径或用 `join` |
| 分隔符不是逗号 | `read_csv(file)` | 列读成一整列 | `pd.read_csv(file, sep=";")` |
| 文件太大 | 一次全读 | 内存爆 | `pd.read_csv(file, nrows=1000)` 先试 |

## 调试顺序
