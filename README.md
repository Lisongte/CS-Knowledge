# CS-Knowledge

这是我的 CS 之路知识记录库，用来整理我在学习计算机过程中的笔记、代码理解、问题复盘和阶段性总结。

## 为什么记录

学习计算机不是只靠“看懂”就够了，更重要的是把知识变成可以复用的理解。这个仓库会记录：

- 新学到的概念
- 重要课程和书籍笔记
- 编程实践中的问题与解决方案
- 算法、系统、网络、数据库等专题总结
- 面试、项目和长期学习路线的复盘

## 目录规划

这个知识库会按主题分区，每个大板块都有自己的入口页：

```text
CS-Knowledge/
├── README.md
├── programming/        # 编程语言与工程实践
├── algorithms/         # 数据结构与算法
├── systems/            # 操作系统、计算机组成、编译原理
├── networks/           # 计算机网络
├── databases/          # 数据库
├── machine-learning/   # 机器学习、Kaggle、数据分析
├── projects/           # 项目记录与复盘
└── reflections/        # 学习总结、计划与反思
```

## 记录方式

每篇笔记尽量回答三个问题：

1. 这个知识点是什么？
2. 它为什么重要？
3. 我能不能用自己的话解释，并用例子验证？

推荐的单篇笔记结构：

```md
# 标题

## 核心概念

## 我的理解

## 示例

## 易错点

## 参考资料
```
