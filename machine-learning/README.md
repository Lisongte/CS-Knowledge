# 机器学习

这里记录机器学习、数据分析、Kaggle 实战和相关 Python 工具的学习笔记。

机器学习板块不要只堆模型公式，也要记录完整工作流：数据在哪里、怎么读入、怎么看数据、怎么验证、怎么提交。很多 Kaggle 题目的关键不是一上来调复杂模型，而是先把数据处理链路跑顺。

## 推荐记录结构

每一篇机器学习笔记可以尽量包含：

- 场景：这个知识点通常在什么任务中出现
- 代码：最小可运行示例
- 解释：每一行代码在做什么
- 常见错误：路径、缺失值、维度、数据泄露等问题
- 实战习惯：在 Kaggle notebook 或本地环境里怎么写更稳

## 当前学习重点

近期可以围绕 Kaggle 建立基本流程：

1. 读取数据
2. 查看数据形状和字段
3. 做简单 EDA
4. 处理缺失值和类别特征
5. 训练一个 baseline 模型
6. 生成 submission 文件
7. 根据验证集和排行榜结果复盘
